# 华恒科技企业介绍PPT网站 - 离线版

## ⚠️ 重要提示

**本网站使用了现代Web技术（ES模块、WebGL），需要通过本地服务器访问才能正常运行。**

直接双击 `index.html` 会因为浏览器安全策略而显示空白页面。

---

## 🚀 快速开始

### Windows 用户

**方法一：双击启动脚本（推荐）**
1. 解压ZIP文件
2. 双击 `快速启动(Windows).bat`
3. 脚本会自动检测环境并启动服务器
4. 自动打开浏览器访问

**方法二：使用Node.js**
1. 安装 [Node.js](https://nodejs.org/)（如果未安装）
2. 双击 `server.js` 或在命令行运行 `node server.js`
3. 浏览器访问 http://localhost:8080

**方法三：使用Python**
1. 安装 [Python](https://www.python.org/)（如果未安装）
2. 打开命令提示符，进入文件夹
3. 运行 `python -m http.server 8080`
4. 浏览器访问 http://localhost:8080

---

### Mac 用户

**方法一：双击启动脚本**
1. 右键点击 `启动.command`
2. 选择"打开"
3. 自动启动服务器

**方法二：使用Python**
1. 打开终端，进入文件夹
2. 运行 `python3 -m http.server 8080`
3. 浏览器访问 http://localhost:8080

---

### Linux 用户

```bash
cd 文件夹路径
python3 -m http.server 8080
```

然后浏览器访问 http://localhost:8080

## ❓ 常见问题

### Q: 为什么直接双击 index.html 打不开？
**A:** 网站使用了 ES 模块技术，浏览器的安全策略禁止本地文件 (`file://` 协议) 直接加载模块。必须通过本地服务器 (`http://localhost`) 访问。

### Q: 如何安装Python？
**A:** 
1. 访问 https://www.python.org/
2. 下载最新版本
3. 安装时**务必勾选** "Add Python to PATH"
4. 安装完成后重新运行启动脚本

### Q: 3D星球效果不显示？
**A:** 
- 确保浏览器支持 WebGL
- 尝试更新显卡驱动
- 或使用在线版本：https://nyrjlixoiayhs.ok.kimi.link

### Q: 启动脚本提示找不到Python/Node.js？
**A:** 
- 先安装 Python 或 Node.js
- 安装后可能需要重启电脑
- 或直接使用在线版本
